# План разработки h-matrix

## Стек
- HTML5 + CSS3 + Vanilla JS
- Bootstrap 5 (CDN)
- File System Access API для работы с файлами

## Структура проекта
```
hmatrix/
├── index.html          # Главный экран с матрицей
├── css/
│   └── styles.css      # Кастомные стили
├── js/
│   ├── app.js          # Основная логика приложения
│   ├── matrix.js       # Логика матрицы и квадрантов
│   ├── task.js         # Класс Task и парсинг/сериализация
│   ├── file-handler.js # Работа с файлами (открытие/сохранение)
│   ├── modal.js        # Модальное окно создания/редактирования
│   └── views/          # Модули видов
│       ├── ViewRenderer.js  # Базовый класс для видов
│       ├── MatrixView.js    # Матрица (4 квадранта)
│       ├── ListView.js      # Список (группировка)
│       └── ColumnView.js    # Колонки (вертикальные)
├── tasks/
│   └── *.md            # Примеры задач (1 задача = 1 файл)
├── plan.md             # Этот файл
└── README.md           # Документация
```
