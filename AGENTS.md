# h-matrix — Спецификация для AI-агентов

## Быстрый контекст

**Стек:** Vanilla JS + Bootstrap 5 (CDN)
**Статус:** ✅ Завершён (v0.2.6)

## Правила работы

**Не обновлять версию приложения самостоятельно** — версия меняется только по явному запросу пользователя.

**Не писать техническую документацию в AGENTS.md** — выносить в отдельные `.md` файлы в папке `docs/`.

AGENTS.md должен содержать только:
- Краткую справку о стеке и статусе
- Ссылки на основную документацию
- Структуру проекта с описанием модулей
- Ключевые точки расширения

Подробная техническая документация (API, алгоритмы, настройки) должна быть в отдельных файлах в `docs/`.

## Документация

- **README.md** — пользовательская документация, UI/UX, формат задач
- **plan.md** — план разработки, статус этапов, история реализации
- **docs/overlay.md** — startup overlay: архитектура, API, алгоритмы, настройки
- **css/variables.css** — справочник всех цветов приложения (CSS-переменные)
- **css/overlay.css** — стили startup overlay (воронка, иконки, анимации)

## Архитектура

```
h-matrix/
├── index.html
├── css/
│   ├── variables.css      # CSS-переменные для всех цветов (единый справочник)
│   ├── styles.css         # Стили приложения (цвета через var(--...))
│   └── overlay.css        # Startup overlay (воронка, мерцающие иконки, анимации)
├── js/
│   ├── app.js             # Инициализация, hotkeys, координация (экспортирует window.App)
│   ├── matrix.js          # Фасад: управление задачами, делегирование видам
│   ├── task.js            # Класс Task, парсинг/сериализация
│   ├── file-handler.js    # File System Access API (открытие папки, чтение/запись)
│   ├── modal.js           # Модальное окно (создание/редактирование/удаление)
│   ├── overlay.js         # Startup overlay менеджер (логотип, кнопка, мерцающие иконки)
│   ├── font-scaler.js     # Масштабирование шрифтов (localStorage, CSS-переменная)
│   └── views/
│       ├── ViewRenderer.js   # Базовый класс для видов (общие утилиты)
│       ├── ColumnView.js     # 4 вертикальные колонки (режим по умолчанию)
│       ├── MatrixView.js     # Матрица 2×2 (4 квадранта)
│       └── ListView.js       # Список с группировкой по квадрантам
├── tasks/              # Пользователь выбирает папку (1 задача = 1 .md файл)
├── plan.md             # История реализации, статус этапов
├── README.md           # Документация пользователя
└── AGENTS.md           # Этот файл
```

## Цветовая схема

Все цвета централизованно управляются через `css/variables.css`. При изменении цветов редактируйте только этот файл.
