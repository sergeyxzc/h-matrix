# h-matrix — Спецификация для AI-агентов

## Быстрый контекст

**Стек:** Vanilla JS + Bootstrap 5 (CDN)
**Статус:** ✅ Завершён (v0.2.2)

## Документация

- **README.md** — пользовательская документация, ограничения платформы, поведение приложения
- **plan.md** — план разработки, статус этапов, история реализации
- **css/variables.css** — справочник всех цветов приложения (CSS-переменные)
Хорошо
## Архитектура

```
h-matrix/
├── index.html
├── css/
│   ├── variables.css      # CSS-переменные для всех цветов (единый справочник)
│   └── styles.css         # Стили (все цвета через var(--...))
├── js/
│   ├── app.js             # Инициализация, hotkeys, координация
│   ├── matrix.js          # Фасад: управление задачами, делегирование видам
│   ├── task.js            # Класс Task, парсинг/сериализация
│   ├── file-handler.js    # File System Access API
│   ├── modal.js           # Модальное окно (создание/редактирование)
│   └── views/
│       ├── ViewRenderer.js   # Базовый класс для видов (общие утилиты)
│       ├── ColumnView.js     # 4 вертикальные колонки (режим по умолчанию)
│       ├── MatrixView.js     # Матрица 2×2 (4 квадранта)
│       └── ListView.js       # Список с группировкой по квадрантам
├── tasks/              # Пользователь выбирает папку (1 задача = 1 .md файл)
├── plan.md             # История реализации, статус этапов
├── README.md           # Документация пользователя
└── AGENTS.md           # Этот файл
```

## Цветовая схема

Все цвета централизованно управляются через `css/variables.css`. При изменении цветов редактируйте только этот файл.
