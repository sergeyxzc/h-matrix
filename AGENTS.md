# h-matrix — Спецификация для AI-агентов

## Быстрый контекст

**Стек:** Vanilla JS + Bootstrap 5 (CDN)
**Статус:** ✅ Завершён (v0.2.5)

## Документация

- **README.md** — пользовательская документация, UI/UX, формат задач
- **plan.md** — план разработки, статус этапов, история реализации
- **css/variables.css** — справочник всех цветов приложения (CSS-переменные)
- **css/overlay.css** — стили startup overlay (воронка, иконки, анимации)

## Архитектура

```
h-matrix/
├── index.html
├── css/

│   ├── task.js            # Класс Task, парсинг/сериализация

│   ├── overlay.js         # Startup overlay менеджер (логотип, кнопка, мерцающие иконки)

│       ├── ViewRenderer.js   # Базовый класс для видов (общие утилиты)
```

## Модули

### overlay.js

**OverlayManager** (singleton, экспортируется в `window.overlayManager`)

Управляет startup overlay экраном:
- Показывает overlay при загрузке страницы
- Генерирует мерцающие иконки тайм-менеджмента (макс. 3 одновременно)
- Анимация: плавное появление/исчезновение (4-12 сек), случайные позиции в 16 зонах экрана
- Вращающаяся спиральная воронка из 5 колец (60 сек/оборот)
- Логотип в центре — кликабелен, вызывает `App.openFolder()`
- Кнопка "Начать работу" — появляется через 0.5 сек, скрывает overlay

**Методы:**
- `startLoading()` — запуск анимации (вызывается при инициализации)
- `hide()` — плавное скрытие overlay (0.8 сек)
- `hideImmediately()` — мгновенное скрытие
- `show()` — показать overlay (для отладки)

**Точки расширения:**
- Изменение количества иконок: `this.maxIcons = N` в конструкторе
- Изменение частоты появления: `scheduleNextIcon()` (1-3 сек рандом)
- Добавление новых типов иконок: массив `icons` в `startFloatingIcons()`


## Цветовая схема

Все цвета централизованно управляются через `css/variables.css`. При изменении цветов редактируйте только этот файл.
